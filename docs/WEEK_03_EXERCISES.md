# Week 3 Exercises - Advanced Features

## Exercise 1: Caching Implementation (40 minutes)

**Task:** Add caching to a new endpoint

**Requirements:**
1. Choose an endpoint that could benefit from caching
2. Add `@cache_result` decorator
3. Test cache hits/misses
4. Measure performance improvement

**Deliverable:** Cached endpoint with benchmarks

---

## Exercise 2: Performance Profiling (30 minutes)

**Task:** Profile system performance

**Requirements:**
1. Use Python profiler or timing
2. Measure:
   - Classification time
   - Vector search time
   - Database query time
3. Identify bottlenecks
4. Suggest optimizations

**Deliverable:** Performance profile report

---

## Exercise 3: Database Migration Plan (25 minutes)

4. Identify compatibility issues

**Deliverable:** Migration plan document

## Exercise 4: Unit Tests (45 minutes)

**Task:** Write comprehensive unit tests

**Requirements:**
1. Test at least 3 functions
2. Include:
   - Normal cases
   - Edge cases
   - Error cases
3. Aim for >80% coverage of your functions

**Deliverable:** Unit test suite
